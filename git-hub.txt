create a new repository on the command line or git bash
1.  git init
2.  git add README.md
3.  git commit -m "first commit"
4.  git -M main
5.  git remote add origin http://github.com//irfanbaihaki84/Basic.git
6.  git push -u origin main

push an existing repository from the command line
1.  git remote add origin https://github.com/irfanbaihaki84/Basic.git
2.  git branch -M main
3.  git push -u origin main

importing code from another repository

pull request from existing repository
git pull https://github.com/irfanbaihaki84/Basic.git

clone request from existing repository
git clone https://github.com/irfanbaihaki84/Basic.git
